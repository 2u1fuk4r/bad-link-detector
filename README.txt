Bad Link Detector
This Python program is used to check if a given URL is bad or malicious. It checks the status code of the URL entered by the user, and if the status code is 400 or higher, the URL is considered bad or malicious. Otherwise, the URL is considered safe.

====How to Use?====
To run the program, type the following command in your terminal or command prompt: python bad_link_detector.py

The program will prompt you to enter a URL. After entering the URL, the program will check if the URL is bad or malicious and display the result on the screen.

Please note that when entering the URL, you need to use the "http" or "https" prefix. Otherwise, the program may not produce the correct results.

=====Requirements=======
The program requires the following Python modules to work:
-requests
To check if these modules are installed, type the following command in your terminal or command prompt: pip show requests
If the module is not installed, you can install it by typing the following command:
pip install requests
